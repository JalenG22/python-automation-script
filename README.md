# python-automation-script
Python scripts for automation.

## Tech
- Python
- Linux
- Git & GitHub

### Scripts
- cleanup_file.py: Organizes files by extension.
- log_parser.py: Parses logs and counts errors.
- system_info.py: Displays system resources usage.
- sample.log: Application log used for testing log parsing.

#### Testing 
Python scripts were tested in GitHub Codespaces (Linux):

- system_info.py: Successfully displayed resource usage
- cleanup_file.py: Successfully organized files
